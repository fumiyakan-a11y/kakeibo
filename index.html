Googleスプレッドシート連携
Apps ScriptのWebアプリURLを入力すると、記録・履歴・分析すべてが「06_[入力用]家計簿」シートと同期されます。
Apps Script 設定手順
1. Googleスプレッドシートを開く
2. 拡張機能 → Apps Script
3. 下のコードを全部貼り付け
4. デプロイ → 新しいデプロイ → ウェブアプリ
5. アクセス：「全員」に設定してデプロイ
6. 発行されたURLを上に入力して保存 
function doPost(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName('06_[入力用]家計簿');
  const data = JSON.parse(e.postData.contents);
  const date = new Date(data.date);
  const days = ['日','月','火','水','木','金','土'];
  sheet.appendRow([
    data.date,
    days[date.getDay()],
    data.category,
    data.item,
    data.cost,
    data.payer
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName('06_[入力用]家計簿');
  const rows = sheet.getDataRange().getValues().slice(1);
  const entries = rows
    .filter(r => r[0])
    .map(r => ({
      date: String(r[0]).substring(0, 10),
      weekday: r[1],
      category: r[2],
      item: r[3],
      cost: Number(r[4]) || 0,
      payer: r[5]
    }));
  return ContentService
    .createTextOutput(JSON.stringify(entries))
    .setMimeType(ContentService.MimeType.JSON);
}
